About the plugin

The AskYourStore WordPress plugin securely connects your WooCommerce store to the AskYourStore AI analytics platform. Once connected, you can ask natural questions about your sales, customers, or products — and get instant answers in Telegram.

Example questions:

  • “How many orders did we have last week?”
  • “Which products sold best in Germany?”
  • “What was our total revenue in July?”

Key features

⚙️ Easy setup — install the plugin, connect Telegram, and start asking.
💬 Natural questions — no SQL or technical skills required.
🛒 WooCommerce ready — works with your store’s data instantly.
🔒 Secure by design — only read-only access via WordPress REST API.

Installation

  1. Download and install the plugin from the official WordPress directory.
  2. Activate it in your WordPress admin panel.
  3. Follow the instructions to connect your store to AskYourStore.
